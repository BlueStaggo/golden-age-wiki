# GenLayerAddSnow *(b1.9-pre+)*

`GenLayerAddSnow` turns 1 in 5 land biomes into ice plains.

## Pseudocode
```lua
if biome != Biome.OCEAN and randbool(5) {
    biome = Biome.ICE_PLAINS
}
```

## Example
World Seed: Glacier<br>
Seed: 20<br>
X: 0<br>
Z: 0<br>
