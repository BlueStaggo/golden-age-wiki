# GenLayerDownfall *(b1.8-r1.1)*

`GenLayerDownfall` copies the downfalls of biomes and puts them in a range between 0 and 65536.

## Pseudocode
```lua
downfall = floor(biome.downfall * 65536)
```

## Example
World Seed: Glacier<br>
Seed: N/A<br>
X: 0<br>
Z: 0<br>
